Treats points within that 1km margin as the 'same spot' to filter out GPS/tower noise.

Identifies 'huge' changes in location—meaning if a user moves to a different neighborhood in the same city (e.g., Tamale), it should be kept as a new location, but repeated pings from the exact same spot should be removed.

import pandas as pd
import numpy as np
from sklearn.cluster import DBSCAN

# 1. Load your data
df = pd.read_csv('246060920_calls_strictly_cleaned.csv')

# 2. Extract coordinates (latitude and longitude)
coords = df[['latitude', 'longitude']].values

# 3. Define the "Distance Margin"
# Here, 1.0 means we group everything within 1 kilometer.
# You can increase this to 2.0 or 5.0 for "hugger" differences.
distance_km = 1.0 
kms_per_radian = 6371.0088
epsilon = distance_km / kms_per_radian

# 4. Run Clustering (DBSCAN) using the Haversine (Earth-curve) formula
db = DBSCAN(eps=epsilon, min_samples=1, algorithm='ball_tree', metric='haversine').fit(np.radians(coords))
